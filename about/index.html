Built a Blockchain Explorer

July - Sept 2022

Summary

I created this explorer for a single purpose - to track users who bypass paying royalties on the swap of assets. Simply put, royalties is a source of passive income paid to creators when an asset they created is traded, bought, or sold.

When a user makes a swap using smart contracts, royalties are programmed into the smart contract directly (so royalties get paid). However, when users do a trade peer-to-peer, there are no royalties in the sale since the royalties aren't programmed on the asset level.

My explorer iterates through a specific wallet address and will look for the swap of NFTs with the collection identifier EAPES-8f3c1f. If such transaction is found, it will then check for the payments of liquid assets to or from the address (this procedure mimics the peer-to-peer transactions). Each detail, such as hashes, senders, and recievers is hyperlinked to the offical explorer for verification purposes.

Skills gained

JavaScript - Asynchronous functions and HTML functionality

APIs - gathered endpoint data
