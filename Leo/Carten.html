My Projects

Blockchain Explorer

A block-chain explorer is something users can use to search the history of any transaction that has ever happened on-chain. My explorer allows users to enter a wallet address, and will return all transactions done in peer-to-peer fashion (without the use of smart-contracts) for a specific on-chain NFT.

A 'make you get out of bed' Alarm Clock

I converted a built-in milliseconds function to seconds, and then displayed the elapsed time on a liquid crystal display - users input at what time they want the alarm to go off. The alarm is programmed to change frequencies, so as the alarm is not shut off, gets more annoying. The 'off' button is located 12 feet away to force you to get out of bed.

Temperature Controlled Oscillating Fan

My program gets the air temperature using a thermistor. If the air temperature is greater than 75 degrees F, the arduino will provide HIGH voltage to the servo motor and fan. The servo motor rotates as the fan blows cool air. The fan will automatically shut off when the temperature is less than or equal to 60 degrees F.

Discord Bot(s)

Discord bots can have many purposes. My bot - written in Python - would request market data from a API endpoints, and then display the floor prices of various NFT collections inside of specific channels in a discord server.

PHP Database Login/Logout Site

I used PHP to connect to my database. Everytime you login, it checks the database to see if the username and password exist. If you don't have an account, you can make one, which adds in a new row with corresponding data into the database. It also includes a logout button which destroys the session and clears cookies.
