Built my own simple programming language

Aug 2023

I was fascinated with how programs actually execute, which lead me to exploring Interpreters, Compilers, and Virtual Machines. After talking to friends, I decided to read Crafting Interpreters (I am still reading this). Crafting Interpreters offers great hands on learning in C and Java, so I decided to follow along in Python - hence creating my own programming language.

The languages name is LOPL, which stands for Leo's Own Programming Language, and uses the Python Virtual Machine for execution. The language can handle printing, variable declaration as strings and integers, arrays, and basic loops.

Reading Crafting Interpreters has offered me one of the greatest learning oppurtunities, and continues to invoke my curiousity.

Temperature Controlled Oscillating Fan

Nov 2021 - Dec 2021

My program gets the air temperature using a thermistor. If the air temperature is greater than 75 degrees F, the arduino will provide HIGH voltage to the servo motor and fan. The servo motor rotates as the fan blows cool air. The fan will automatically shut off when the temperature is less than or equal to 60 degrees F.
